# 🤖 NoteBot Pro

An AI-powered chatbot built using Python, Streamlit, LangChain, and Groq.

## 🚀 Features

- 📄 PDF Question Answering
- 🤖 General AI Chat
- 🔄 Automatic PDF / General Switching
- ✏️ Edit Messages
- 📋 Copy Responses
- 📥 Download Chat
- 🌙 Dark Mode
- 💬 Multiple Chat Sessions
- 📚 Source References

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- Groq API
- PyPDF2

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
